//----------------------------------------------
// HOW TO USE GOOGLE CLOUD SQL TO
// MANIPULATE THE DUDO SQL DATABASE
//----------------------------------------------
- go to Cloud Run:  https://console.cloud.google.com/run
- click hamburger (upper left)
- scroll down to and click on Cloud SQL
- click on dudo-mysql
- left menu:  Cloud SQL Studio
- log in to database:
    - Database:  dudo-db
    - User: dudo-app
    - Password: dudo-app-nodejs
    - Authenticate
- click on tab: '<> Untitled query'

Now you can enter SQL commands and click Run

//----------------------------------------------
// USEFUL SQL SCRIPTS
//----------------------------------------------
// --- see everything in the database ---
SELECT * FROM players;

//--- add a player ---
INSERT INTO players (guid, username, password_hash)
VALUES (
  UUID(),
  'glenn',
  '$2b$10$TmB/vW8IIM./q0BCyh2OsuGRdsB04ZFPqEWPnKP1bPMMVW6bSyymi'
);
NOTE: You first have to encrypt (hash) the password.
from the VSCode terminal, something like:
cmd /c hash mypassword
It will generate an encrypted password, which you paste into the command above.

// --- delete a player ---
DELETE FROM players
WHERE username = 'glenn';

still in the google cloud console:
- click dudo-mysql
- click Cloud SQL Studio
- put in credentials
- click New SQL editor tab
- put in the editor:
    CREATE TABLE players (
    id INT AUTO_INCREMENT PRIMARY KEY,
    guid CHAR(36) NOT NULL UNIQUE,
    username VARCHAR(50) NOT NULL UNIQUE,
    password_hash VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
    );
